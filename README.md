# sieve-web

The official website for **Sieve** — a fully local LLM-traffic security proxy
for crypto-native developers. Served via GitHub Pages at
**[sieveai.dev](https://sieveai.dev)**.

Sieve is a single Rust binary that sits on `127.0.0.1` between your AI coding
agent (Claude Code / OpenClaw / Hermes / Codex CLI) and the upstream model API
(Anthropic / OpenAI / relays). It inspects traffic in both directions —
redacting secrets on the way out, intercepting dangerous tool calls on the way
in (fail-closed) — so there is a moment of cognitive friction before
irreversible actions (signing, transfers, deploys).

The visual design was produced in [Claude Design](https://claude.ai/design) and
ported here to plain, dependency-free static HTML.

## Design constraints

- **Self-contained static HTML, zero external runtime dependencies.** No
  framework, **no CDN** — including fonts: JetBrains Mono / IBM Plex Sans /
  IBM Plex Mono are **self-hosted** under `assets/fonts/` (a privacy product
  must not phone home to Google Fonts). Build scripts under `scripts/` run
  manually before commit; the served site stays plain static files.
- **Theming:** light / dark / system, driven by `data-theme` on `<html>`;
  preference persists in `localStorage` (`sieve.theme`).
- **Bilingual, split URLs (SEO):** English at `/`, 简体中文 under `/zh/`, with
  self-referencing canonicals + `hreflang` cross-links. Both versions are
  generated from single-source pages in `src/` by `scripts/build_site.py`
  (Python stdlib only); the language switcher is a plain link, never a
  JS toggle or auto-redirect.
- **Responsive** and accessible (focus-visible, ARIA, `prefers-reduced-motion`).

## Directory layout

```
sieve-web/
├── src/            # Single-source bilingual pages (edit THESE, not the generated output)
│   ├── index.html  #   landing (hero, how-it-works, detection, verify, get-started)
│   └── docs.html   #   documentation hub (sidebar + quickstart + cards)
├── index.html      # GENERATED — English landing (do not edit by hand)
├── docs.html       # GENERATED — English docs hub
├── zh/             # GENERATED — 简体中文 versions (/zh/, /zh/docs.html)
├── 404.html        # Friendly bilingual not-found page (noindex, not generated)
├── scripts/
│   ├── build_site.py    # src/*.html → en + zh outputs + sitemap.xml (stdlib only)
│   └── build_assets.py  # og.png / favicon.ico / apple-touch-icon.png (Pillow, build-time)
├── assets/
│   ├── style.css   # Shared design system (themes, tokens, components, animations)
│   ├── app.js      # Theme controls, animated sequence diagram, scroll reveal, mobile nav
│   ├── og.png      # Social share card (1200×630)
│   └── fonts/      # 11 self-hosted woff2 (JetBrains Mono / IBM Plex Sans / IBM Plex Mono)
├── favicon.ico     # 48/32/16 multi-size (Google Search needs a fetchable ≥48px favicon)
├── CNAME           # GitHub Pages custom domain (sieveai.dev)
├── robots.txt      # Crawl rules + sitemap pointer
├── sitemap.xml     # GENERATED by build_site.py (loc + lastmod)
├── README.md       # This file
├── DEPLOY.md       # GitHub Pages + custom-domain setup
└── docs/           # Published doc pages (quickstart / threat-model / cli, English,
                    #   hand-maintained) + internal design-brief notes
```

## Editing workflow

1. Edit the bilingual sources in `src/` (both languages live side by side as
   `<span class="lang-en">` / `<span class="lang-zh">` pairs — keep parity).
2. Regenerate the outputs and sitemap: `python3 scripts/build_site.py`
3. Commit sources **and** generated files together (GitHub Pages serves the
   repo as-is; there is no CI build).

## Local preview

Pages use root-absolute asset paths, so serve from the repo root:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## 简介（中文）

**Sieve** 官网仓库，通过 GitHub Pages 部署在
**[sieveai.dev](https://sieveai.dev)**。

Sieve 是一个全本地的 LLM 流量安全代理（单个 Rust 二进制），运行在 `127.0.0.1`，
位于你的 AI 编码 agent（Claude Code / OpenClaw / Hermes / Codex CLI）与上游模型 API
（Anthropic / OpenAI / 中转站）之间，双向检测流量：出站脱敏密钥、入站拦截危险
工具调用（fail-closed），在不可逆操作（签名、转账、部署）前制造一道认知摩擦。

视觉设计在 [Claude Design](https://claude.ai/design) 中产出，移植为零依赖静态 HTML。

本站为**自包含静态 HTML、零外部运行时依赖**：无框架、**无 CDN**——字体
（JetBrains Mono / IBM Plex Sans / IBM Plex Mono）也**自托管**于 `assets/fonts/`
（隐私产品不该向 Google Fonts 发起请求）。主题浅色/深色/跟随系统，偏好存于
`localStorage`（`sieve.theme`）。中英双语按独立 URL 拆分（英文在 `/`，中文在
`/zh/`，互挂 hreflang），由 `scripts/build_site.py` 从 `src/` 单源生成——改内容
先改 `src/`，再跑构建脚本，产物一并提交。语言切换是普通链接，不做 JS 切换、
不做自动跳转。

**本地预览**：

```bash
python3 -m http.server 8000
# 打开 http://localhost:8000
```
