DOCUMENTATION文档

Run Sieve in five minutes. 五分钟跑起 Sieve。

Sieve is a local LLM-traffic security proxy — one Rust binary between your coding agent and the model API. These docs cover install, the detection engine, and how to verify every claim yourself. Sieve 是一个本地 LLM 流量安全代理——夹在你的编码 agent 与模型 API 之间的一个 Rust 二进制。这份文档涵盖安装、检测引擎，以及如何亲手验证每一条声明。

Quickstart快速开始
$ cosign verify-blob --key sieve.pub --signature Sieve-1.0.0.dmg.sig Sieve-1.0.0.dmg
$ sieve setup
$ sieve doctor && sieve decisions watch
