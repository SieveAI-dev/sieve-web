五分钟跑起 Sieve。

Sieve 是一个本地 LLM 流量安全代理——夹在你的编码 agent 与大模型 API 之间的一个 Rust 二进制。这份文档涵盖安装、检测引擎，以及如何亲手验证每一条声明。

快速开始
$ cargo build --release # build from source — brew & signed .dmg coming soon
$ sieve setup
$ sieve doctor && sieve decisions watch
